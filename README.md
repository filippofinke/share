<h1 align="center">Welcome to share 👋</h1>

> A simple text sharing based on the connection IP address

### ✨ [Demo](https://share.filippofinke.ch)

## Install

```sh
npm install
```

## Build

```sh
npm run build
```

## Usage

```sh
npm run start
```
